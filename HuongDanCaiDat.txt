================================================================================
                    HUONG DAN CAI DAT - RAGAS Answer Filtering
================================================================================

Du an nay loc cau tra loi RAG theo do trung thuc (faithfulness) voi context
da truy xuat. Pipeline:

  retrieve -> generate -> filter(context, answer) -> accept / reject

Phuong phap chinh: trich dac trung RAGAS (black-box) roi huan luyen bo loc
sklearn. Baseline: LLM-as-judge va DeBERTa / NLI.

--------------------------------------------------------------------------------
1. YEU CAU HE THONG
--------------------------------------------------------------------------------

  - He dieu hanh: Windows / macOS / Linux
  - Python >= 3.10 (khuyen nghi 3.11 hoac 3.12)
  - Ket noi Internet (cai thu vien, goi API OpenAI, tai model HuggingFace)
  - Dung luong o dia: khoang 5 GB (moi truong + model)

  Tuy chon theo muc dich:

  - Chay RAGAS / LLM-judge: can OPENAI_API_KEY (xem muc 5)
  - Train DeBERTa tren GPU NVIDIA (khuyen nghi):
      * CUDA tuong thich
      * GPU >= 4 GB VRAM (cau hinh batch_size=1 da toi uu cho laptop)
  - CPU van chay duoc DeBERTa/NLI nhung cham hon nhieu

--------------------------------------------------------------------------------
2. CONG CU VA MOI TRUONG PHAT TRIEN
--------------------------------------------------------------------------------

  Cong cu can thiet:
  - Git            : clone ma nguon
  - Python / venv  : moi truong ao (hoac Conda)
  - pip            : cai thu vien Python
  - Jupyter / VS Code / Cursor : chay notebook (tuy chon)
  - Trinh duyet    : tai models/ tu Google Drive (tuy chon)

  Moi truong ao (khuyen nghi):
  - Ten goi y: ragas-eval
  - Python trong moi truong: 3.10+

--------------------------------------------------------------------------------
3. CAI DAT BUOC BO
--------------------------------------------------------------------------------

  Buoc 1. Clone / mo thu muc du an

      git clone https://github.com/Htaaxx/ragas-evaluation.git
      cd ragas-evaluation

      (Neu da co san thu muc ma nguon thi chi can cd vao thu muc goc
       — thu muc chua file HuongDanCaiDat.txt nay.)

  Buoc 2. Tao moi truong ao

      python -m venv venv

      Windows:
          venv\Scripts\activate

      Linux / macOS:
          source venv/bin/activate

      (Hoac Conda: conda create -n ragas-eval python=3.11 -y
       roi conda activate ragas-eval)

  Buoc 3. Cai thu vien Python

      pip install -r requirements.txt

      Neu train DeBERTa tren GPU NVIDIA (tuy chon):

          pip install torch --index-url https://download.pytorch.org/whl/cu124

  Buoc 4. Cau hinh API key

      - Tao file .env o thu muc goc du an (khong co file mau .env_template).
      - Them it nhat:

          OPENAI_API_KEY=sk-...

      - Tuy chon (neu notebook doc toi):

          HUGGINGFACE_API_KEY=...
          GOOGLE_API_KEY=...

      - KHONG commit file .env co khoa that len Git.

  Buoc 5. (Tuy chon) Tai thu muc models/

      Neu can dung checkpoint da train san, tai thu muc models/ tu Google Drive:

          https://drive.google.com/drive/folders/1ZrcY2-_Llr5GJSeZESIXcUmcUWDfjjBc?usp=sharing

      Dat models/ ngay duoi thu muc goc du an (cung cap voi src/, notebooks/, data/).
